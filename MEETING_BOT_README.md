# 🤖 Telegram Bot для автоматизации встреч

Этот бот автоматически присоединяется к онлайн-встречам, записывает аудио, транскрибирует его, анализирует через Gemini AI и обновляет лиды в Bitrix24.

## 🚀 Возможности

- **Автоматическое присоединение к встречам** на платформах:
  - Zoom
  - Google Meet
  - Microsoft Teams
  - Kontur Talk
  - Yandex Telemost

- **Запись и транскрипция аудио** с использованием Whisper AI
- **Анализ встреч** через Gemini AI с использованием чек-листов
- **Автоматическое обновление лидов** в Bitrix24 CRM
- **Создание задач** на основе анализа встреч

## 📋 Требования

### Системные требования
- Python 3.8+
- Chrome/Chromium браузер
- PulseAudio (для Linux)
- FFmpeg (для обработки аудио)

### Python зависимости
```bash
pip install -r requirements.txt
```

### Переменные окружения
Создайте файл `.env` на основе `.env.example`:

```env
# Telegram Bot
TELEGRAM_BOT_TOKEN=your_bot_token
ADMIN_CHAT_ID=your_admin_chat_id

# Gemini AI
GEMINI_API_KEY=your_gemini_api_key
GEMINI_MODEL=gemini-1.5-flash

# Bitrix24
BITRIX_WEBHOOK_URL=your_bitrix_webhook_url
BITRIX_RESPONSIBLE_ID=your_responsible_id

# Настройки
MEETING_DURATION_MINUTES=30
AUDIO_QUALITY=high
```

## 📱 Использование

### Команды бота

- `/start` - Начать работу с ботом
- `/help` - Показать справку
- `/status` - Проверить статус бота

### Рабочий процесс

1. **Отправьте ссылку на встречу** боту
2. Бот автоматически:
   - Определит платформу встречи
   - Присоединится к встрече
   - Запишет аудио
   - Транскрибирует речь
   - Проанализирует встречу по чек-листу
3. **Получите анализ** в Telegram чате
4. **Отправьте ID лида** в Bitrix24
5. Бот обновит лид и создаст задачи

## 🔧 Архитектура

### Основные компоненты

- **`main.py`** - Главный файл бота, обработка сообщений
- **`meeting_link_processor.py`** - Оркестратор процесса встреч
- **`aggressive_meeting_automation.py`** - Автоматизация присоединения к встречам
- **`speech_transcriber.py`** - Транскрипция аудио через Whisper
- **`meeting_analyzer.py`** - Анализ встреч с чек-листами
- **`gemini_client.py`** - Интеграция с Gemini AI
- **`bitrix.py`** - Интеграция с Bitrix24
- **`bitrix_meeting_integration.py`** - Обновление лидов на основе анализа

### Поток данных

```
Ссылка на встречу → Определение платформы → Присоединение → 
Запись аудио → Транскрипция → Анализ через Gemini → 
Отправка в Telegram → Запрос ID лида → Обновление Bitrix24
```

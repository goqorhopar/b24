# 🤖 АВТОНОМНЫЙ СЕРВЕРНЫЙ БОТ

## 🎯 ЦЕЛЬ
Бот работает **ПОЛНОСТЬЮ АВТОНОМНО** на сервере **БЕЗ ВАШЕГО УЧАСТИЯ**!

## 🚀 ВАРИАНТЫ ДЕПЛОЯ

### 1️⃣ SYSTEMD СЕРВИС (Рекомендуется)

```bash
# На сервере Linux
sudo ./deploy_autonomous_server.sh
```

**Что происходит:**
- ✅ Бот устанавливается как системный сервис
- ✅ Автоматический запуск при перезагрузке сервера
- ✅ Автоматический перезапуск при сбоях
- ✅ Работает в фоне без вашего участия

### 2️⃣ DOCKER КОНТЕЙНЕР

```bash
# На любом сервере с Docker
./deploy_docker_autonomous.sh
```

**Что происходит:**
- ✅ Бот упаковывается в Docker контейнер
- ✅ Автоматический перезапуск при сбоях
- ✅ Изолированная среда выполнения
- ✅ Легкое масштабирование

## 🔧 УПРАВЛЕНИЕ

### Systemd сервис:
```bash
systemctl status meeting-bot-autonomous.service    # Статус
systemctl restart meeting-bot-autonomous.service   # Перезапуск
journalctl -u meeting-bot-autonomous.service -f    # Логи
systemctl stop meeting-bot-autonomous.service      # Остановка
```

### Docker контейнер:
```bash
docker ps                                          # Статус
docker logs autonomous-meeting-bot -f              # Логи
docker restart autonomous-meeting-bot              # Перезапуск
docker stop autonomous-meeting-bot                 # Остановка
```

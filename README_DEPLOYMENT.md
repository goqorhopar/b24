# 🚀 Развертывание Telegram Бота для Автоматизации Встреч

## 📋 Обзор
Этот бот автоматически присоединяется к встречам на различных платформах (Zoom, Google Meet, Teams, Kontur.Talk, Yandex Telemost), записывает аудио и анализирует контент с помощью ИИ.

## 🔧 Основные Исправления

### ✅ Исправленные Проблемы:
1. **Бесконечный цикл аудиозаписи** - добавлена проверка `is_recording`
2. **Проблемы с аудиоустройствами** - улучшен выбор устройств PulseAudio
3. **Импорты и зависимости** - исправлены все импорты модулей
4. **Платформы встреч** - добавлена поддержка всех платформ
5. **Агрессивная автоматизация** - бот пробует все возможные способы присоединения

## 📁 Структура Файлов

### Основные файлы:
- `main.py` - главный файл Flask приложения
- `aggressive_meeting_automation.py` - агрессивная автоматизация встреч
- `meeting_link_processor.py` - обработчик ссылок на встречи
- `platform_detector.py` - детектор платформ встреч

### Скрипты развертывания:
- `FIX_AUDIO_LOOP.sh` - исправление бесконечного цикла аудиозаписи
- `FINAL_VERIFICATION.sh` - финальная проверка всех компонентов
- `CHECK_BOT_STATUS.sh` - проверка статуса бота

## 🐛 Устранение Неполадок

### Бот не присоединяется к встрече:
1. Проверьте логи: `journalctl -u telegram-bot -f`
2. Убедитесь, что ссылка корректная
3. Проверьте доступность платформы

### Проблемы с аудиозаписью:
1. Проверьте PulseAudio: `pactl list sources`
2. Установите pulseaudio-utils: `sudo apt install pulseaudio-utils`
3. Перезапустите бота: `sudo systemctl restart telegram-bot`

### Ошибки импорта:
1. Очистите кеш Python: `sudo find /opt/telegram-bot -name "*.pyc" -delete`
2. Перезапустите бота
3. Проверьте права доступа к файлам

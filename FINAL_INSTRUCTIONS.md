# 🎯 ФИНАЛЬНЫЕ ИНСТРУКЦИИ ДЛЯ ЗАПУСКА БОТА

## ⚠️ КРИТИЧЕСКИ ВАЖНО:
**Бот должен работать ТОЛЬКО на Linux сервере!**
**На Windows бот НЕ БУДЕТ работать с встречами!**

---

## 🚀 ШАГИ ДЛЯ ЗАПУСКА:

### 1. Подключитесь к вашему Linux серверу:
```bash
ssh ваш_пользователь@ваш_сервер_ip
```

### 2. Загрузите файлы бота на сервер:
```bash
# Скопируйте все файлы проекта в папку на сервере
# Убедитесь, что .env файл содержит все настройки
```

### 3. Запустите бота:
```bash
# Перейдите в папку с ботом
cd /path/to/bot/folder

# Сделайте скрипт исполняемым
chmod +x quick_start_server.sh

# Запустите бота
./quick_start_server.sh
```

### 4. Проверьте работу:
```bash
# Проверьте процессы
ps aux | grep python

# Проверьте логи
tail -f bot.log
```

## 🆘 ЕСЛИ БОТ НЕ ПРИСОЕДИНЯЕТСЯ К ВСТРЕЧЕ:

### Проверьте:
1. **Логи:** `tail -f bot.log`
2. **Chrome/Chromium установлен:** `google-chrome --version`
3. **Аудио система:** `pactl list sources`
4. **Права доступа:** `ls -la start_bot_fixed.py`

### Возможные проблемы:
- ❌ Chrome не установлен
- ❌ Нет прав на запись аудио
- ❌ Проблемы с сетью
- ❌ Неправильные настройки в .env

---

## 📞 ПОДДЕРЖКА:

Если бот не работает:
1. Проверьте логи: `tail -f bot.log`
2. Проверьте статус: `ps aux | grep python`
3. Перезапустите: `pkill -f python && ./quick_start_server.sh`

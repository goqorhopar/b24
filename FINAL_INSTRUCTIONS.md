# 🚀 ФИНАЛЬНЫЕ ИНСТРУКЦИИ ДЛЯ ЗАПУСКА БОТА

## ❌ Проблема с PowerShell

PowerShell на Windows не может запустить Python скрипты из-за ошибок кодировки.

## ✅ РЕШЕНИЕ: Используйте командную строку Windows (cmd)

### 1. Откройте командную строку Windows (cmd):
- Нажмите `Win + R`
- Введите `cmd`
- Нажмите Enter

### 2. Перейдите в папку с ботом:
```cmd
cd C:\Users\PC\Downloads\гитхаб\b24
```

### 3. Запустите бота:
```cmd
START_BOT.cmd
```

ИЛИ напрямую:
```cmd
python autonomous_bot.py
```

## 🎯 Что произойдет:

1. ✅ Бот запустится
2. ✅ Подключится к Telegram API
3. ✅ Начнет получать сообщения
4. ✅ Будет отвечать на команды
5. ✅ Будет анализировать встречи через Gemini AI
6. ✅ Будет обновлять лиды в Bitrix24

## 📱 Проверьте в Telegram:

Отправьте боту:
- `/start` - начать работу
- `/status` - статус системы
- Ссылку на встречу (например: `https://meet.google.com/test`)

## 🚨 ВАЖНО:

**БОТ ДОЛЖЕН ЗАПУСКАТЬСЯ ЧЕРЕЗ CMD, НЕ ЧЕРЕЗ POWERSHELL!**

**PowerShell вызывает ошибки кодировки и не может запустить Python скрипты!**

## 🔧 Если не работает:

1. Убедитесь что Python установлен
2. Убедитесь что установлены зависимости: `pip install requests google-generativeai`
3. Запускайте через cmd, не через PowerShell
4. Проверьте логи в `logs/autonomous_bot.log`

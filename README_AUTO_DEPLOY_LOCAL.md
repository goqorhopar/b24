# 🚀 Автоматический деплой для локального репозитория

## 🎯 Что настроено

Создана **полностью автоматизированная система** для вашего локального репозитория `goqorhopar/b24`:

- ✅ **Автоматический коммит** при любых изменениях
- ✅ **Автоматический деплой** в GitHub при каждом push
- ✅ **Отслеживание файлов** в реальном времени
- ✅ **Git hooks** для автоматизации
- ✅ **Уведомления** в Telegram

## ⚡ БЫСТРЫЙ СТАРТ

### 1. Заполните .env файл

```bash
# Скопируйте пример конфигурации
copy env_example.txt .env

# Отредактируйте настройки
notepad .env
```

**Обязательные настройки:**
```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
GEMINI_API_KEY=ваш_ключ_gemini
GITHUB_REPO=goqorhopar/b24
GITHUB_TOKEN=ваш_github_токен
DEPLOY_SERVER_URL=ip_вашего_сервера
DEPLOY_SERVER_USER=пользователь_сервера
```

### 2. Запустите автоматический деплой

```bash
# Windows
start_auto_deploy.bat

# Или вручную
python file_watcher.py
```

## 🔄 Как это работает

### Автоматический процесс:

1. **📝 Изменения в файлах** → Отслеживание каждые 10 секунд
2. **🔄 Автоматический коммит** → `git add .` + `git commit`
3. **🚀 Автоматический деплой** → `python auto_deploy.py --commit`
4. **📤 Push в GitHub** → Отправка в `goqorhopar/b24`
5. **🖥️ Деплой на сервер** → Автоматическое развертывание
6. **📱 Уведомления** → Сообщения в Telegram

### Git Hooks:

- **Pre-push hook** - автоматически запускает деплой перед push
- **Post-commit hook** - отправляет уведомления после коммита

## 🛠️ Команды

### Основные команды

```bash
# Запуск автоматического деплоя
start_auto_deploy.bat          # Windows
./start_auto_deploy.sh         # Linux/Mac

# Ручной коммит и деплой
auto_commit.bat                # Windows
./auto_commit.sh               # Linux/Mac

# Отслеживание файлов
python file_watcher.py

# Полный деплой
python auto_deploy.py
```

### Проверка статуса

```bash
# Статус git
git status

# Последние коммиты
git log --oneline -5

# Статус сервиса на сервере
ssh user@server 'systemctl status meeting-bot'
```

## 📁 Созданные файлы

### Скрипты автоматизации
- `file_watcher.py` - отслеживание изменений файлов
- `auto_commit.sh` / `auto_commit.bat` - автоматический коммит
- `start_auto_deploy.sh` / `start_auto_deploy.bat` - запуск системы
- `auto_deploy.py` - основной скрипт деплоя

### Git Hooks
- `.git/hooks/pre-push` - автоматический деплой перед push
- `.git/hooks/post-commit` - уведомления после коммита

### Конфигурация
- `env_example.txt` - пример настроек (ОБНОВЛЕН)
- `.github/workflows/auto-deploy.yml` - GitHub Actions

## 🎯 Использование

### Обычная работа:

1. **Редактируйте файлы** как обычно
2. **Система автоматически**:
   - Обнаруживает изменения
   - Создает коммит
   - Деплоит в GitHub
   - Развертывает на сервер
   - Отправляет уведомления

### Ручной деплой:

```bash
# Если нужно принудительно задеплоить
python auto_deploy.py --commit
```


```bash

```

